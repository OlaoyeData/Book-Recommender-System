# Book-Recommender-System
End-to-end Book Recommendation System that analyzes user preferences and book metadata to suggest relevant titles. The project covers data preprocessing, feature engineering, model development, evaluation, and deployment, demonstrating a complete machine learning pipeline for personalized recommendations.

# How to Run?
### STEPS:

Clone the respository

```bash
https://github.com/OlaoyeData/Book-recommender-System.git
```

### STEP 01- Create a conda environment after opening the repository

```bash
conda create -n books python -y
```

```bash
conda activate books
```

### STEP 02- Install the requirements
```bash
pip install -r requirements.txt
```
